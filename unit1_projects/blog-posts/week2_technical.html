Good Websites: an Analysis

Padding, borders, margins? What's going on?

Learning the difference between these is not overly hard, although it may seem a bit funky at first.
Let's imagine we have something to write, we'll have to put in a paragraph. This is really the same for all elements, as they all have a 2 dimensional box shape.

Ok, so now we have some words. So what? It doesn't look cool. I say we should add a border

Ok, so that's what a border is. Nice, but it's ugly! If only it were further away from the words, it would look way better. Let's put some PADDING on it

So, now you can see what padding is. It's the distance between the content of the element and the border. The border marks the outer boundries of the element, and everything outside of it is another element entirely.
Let's put another element in there. I like text, so that will do.

WOAH! That's fine and all, but the new element is basically touching my shiney new border. We can't have that! We'll have to add a margin

Good! Now nothing's getting near my first element. So there you have it:

  • The border marks the outer boundary of the element.
  • The padding marks an inner boundary, between the border and the contents of the element. In this example, it was the text inside a box-shaped element
  • And finally the margin extends beyond the boundary(/border) of the element. This is used to make space between elements, and is invisible.
